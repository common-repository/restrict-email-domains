== Description ==

Plugin to restrict email domain while user registering.

Major features include:

* Block certain email domains from registering.
* Add a list of allowed email domains to register.

== Installation ==

Upload the plugin to your blog, activate it, and then add the allowed domains in the Email Domains section in the dashboard.

You're done!
